# Old-Fruman-Dictionary
Expanding lookup for words in Old Fruman

This dictionary uses the agreed upon standard in the docs for this language, which is to say:
C = SH
D = TH
V = AZ
W = NII
X = UTH
Y = TOL
Z = EK

Entering characters into the entry box will filter the results, showing words that start with what you've entered.

The add word button will create a window with the word in the textbox already in the entry box. this word can be modified after the window is created.

Each entry has an edit option allowing you to edit the translation, but not the word itself.
Each entry also has a button to delete it if you accidently add a miss-typed word.

When you have modified the dictionary make a push request with the JSON file to this github page, and it should go through. If you can't do this send the json file to me on discord (Myst3350) and i'll commit the changes myself.
